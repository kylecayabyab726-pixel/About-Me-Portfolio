Programming Languages

<3>HTML

HTML is used for creating the structure
of websites. By structure, I mean the
elements of HTML, such as paragraphs
lists, headings, and the addition of
images and videos. These elements help
create the foundation of a webpage, which
can then be styled using CSS.

CSS


CSS is used for styling the layouts and
structure of a website. It allows you to
change fonts, colors, and change text sizes,
as well as control spacing, backgrounds, and
designs. CSS can also be used to position
content on the website, making websites more
appealing to the audience.
